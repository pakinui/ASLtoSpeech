\doxysubsection{Detailed Description}
This is the base class for all reporters.

If are writing a reporter, you must derive from this type, or one of the helper reporter bases that are derived from this type.

\mbox{\hyperlink{class_catch_1_1_reporter_base}{Reporter\+Base}} centralizes handling of various common tasks in reporters, like storing the right stream for the reporters to write to, and providing the default implementation of the different listing events.
