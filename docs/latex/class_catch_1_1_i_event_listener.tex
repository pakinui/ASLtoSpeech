\doxysubsection{Detailed Description}
The common base for all reporters and event listeners

Implementing classes must also implement\+: \begin{DoxyVerb}//! User-friendly description of the reporter/listener type
static std::string getDescription()
\end{DoxyVerb}
 Generally shouldn\textquotesingle{}t be derived from by users of Catch2 directly, instead they should derive from one of the utility bases that derive from this class.
