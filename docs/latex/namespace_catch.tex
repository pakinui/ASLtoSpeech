Calculates the ULP distance between two floating point numbers

The ULP distance of two floating point numbers is the count of valid floating point numbers representable between them.

There are some exceptions between how this function counts the distance, and the interpretation of the standard as implemented. by e.\+g. {\ttfamily nextafter}. For this function it always holds that\+:
\begin{DoxyItemize}
\item {\ttfamily (x == y) =$>$ ulp\+Distance(x, y) == 0} (so {\ttfamily ulp\+Distance(-\/0, 0) == 0})
\item {\ttfamily ulp\+Distance(max\+Finite, INF) == 1}
\item {\ttfamily ulp\+Distance(x, -\/x) == 2 $\ast$ ulp\+Distance(x, 0)}
\end{DoxyItemize}
